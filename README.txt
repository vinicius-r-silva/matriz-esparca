Este projeto visa implementar a criação e manipulação básica de uma matriz esparsa na linguagem C. Nele pode-se criar uma matriz, atribuir, consultar e modificar valores, calcular a soma de uma determinada linha ou coluna, além de calcular o determinante da matriz, caso essa seja quadrada.

COMPILAÇÃO
* Linux (Obs.: Deve-se conter o GCC instalado) : Acesse a pasta onde estão localizados os arquivos fonte pelo terminal >> execute o comando {} 
* Windows: Pode-se abrir os arquivos usando algum programa como Code Blocks ou Dev-C++ e compilá-lo com o MinGW por eles. Existe ainda a opção de compilar o programa utilizando o prompt de comando: Basta acessar a pasta com os arquivos fonte e usar o mesmo comando do Linux.

EXECUÇÃO
* Linux: Navegue pelo terminal e acesse a pasta do arquivo compilado >> Execute o comando {}
* Windows: Caso se tenha compilado com o MinGW pode-se apenas clicar no arquivo .exe gerado
Se desejar, pode-se acessar a pasta pelo prompt de comando e executar os mesmos comandos do Linux. Há também a opção de utilizar os programas de compilação tais como Code Blocks ou Dev-C++.
